Edit/Undo

This function allows you to undo a single file or folder renaming operation, restoring the original name.  When renaming multiple items using the Mass Rename dialog, all items can be undone as a single operation.

 

    Caution: Explorer++'s Undo cannot undo  a file cut, move or delete.
 
    Info: Renaming a folder which has been bookmarked does not automatically rename the bookmark.
