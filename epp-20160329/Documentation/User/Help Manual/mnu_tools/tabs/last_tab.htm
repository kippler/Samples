Tools/Options.../Close the main window when closing the last tab

System Default:   Yes

Explorer++ usually disallows closure of the last open tab, but with this option enabled, this can be done, and results in Explorer++ closing completely.

With this setting active, you cannot use the close button on the Tab bar; you must use double-click (subject to option), a key combination (Ctrl+W, Ctrl+F4), or a menu item.
